# Fonts

Both fonts are licensed under the SIL Open Font License 1.1 (OFL), which permits
bundling and redistribution.

- **BIZ UDGothic** (`BIZUDGothic-Regular.ttf`) — primary UI font (docs/13 §2).
  Source: https://github.com/googlefonts/morisawa-biz-ud-gothic (OFL.txt included)
- **Noto Emoji** (`NotoEmoji-Regular.ttf`) — monochrome emoji fallback for placeholder
  item icons. Source: Google Fonts "Noto Emoji" (OFL).

TMP font assets (`*SDF.asset`) are dynamic (glyphs rasterised on demand), so no glyph
atlas is baked into the repo.
