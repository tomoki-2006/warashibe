# 09_AWARENESS_PLAYBOOK — 認知獲得の実行計画書
*Status: v3.1 / 「知られなければ、面白さは存在しないのと同じ」。認知をプロダクトと同格の開発対象として扱う*

# 1. ブランドの核（すべての発信の元ネタ）

## 1.1 ワンメッセージ
**「遊んだぶんだけ、お金に強い子になる。」**
- 子ども向け表現: 「わら1本から、せかい一の長者になれる？」
- 学校向け表現: 「地理×経済×共感を、45分で。」

## 1.2 創業ストーリー（PRの弾・取材の答え方の型）
「岡山にルーツを持つ開発者が、AI時代のものづくりで、日本の昔話を世界水準の金融教育に作り替えた」——①地元性（岡山発・吉備が最初の舞台）②時代性（AIエージェントとの共同開発）③普遍性（誰もが知る昔話）の三本柱で語る。個人の詳細は出しすぎない（プロダクトが主役）。

## 1.3 語彙の統一（メディアキットに明記）
- 使う: 金融教育／お金の教養／交換／もったいない を超えて「いかす」
- 使わない: 投資で儲ける／マネーゲーム／英才教育／ガチャ（存在しないので言及自体しない）

# 2. プリローンチ（Phase 0-1・お金をかけない仕込み）

1. **公式note/ブログ「わらしべ開発記」**: 月2本。設計思想（お金を最初に入れない理由・詰みゼロ設計）を公開。教育関係者と開発者界隈の両方に刺さる骨太記事は、ローンチ時の「文脈のある紹介」を生む種まき
2. **待機リストLP**: 「先行テスト親子50組募集」。テスター募集自体を最初のマーケにする（テスト参加親=初期の伝道者）
3. **X/Instagramの育成**: ぶんの4コマ・特産品トリビア画像を週2。フォロワー数は追わず、ローンチ時に引用できる資産を貯める
4. **教育系メディアへの種まき**: リセマム・こどもとIT・EdTechZine等に「金融教育×ゲームの現在地」の寄稿を打診（宣伝でなく知見提供から入る）

# 3. 主力キャンペーン: 「リアルわらしべチャレンジ」【Phase 2・最重要】

## 3.1 概要
家族で実際にやる「小さなトレードアップ」遊びを公式フォーマット化して流行らせる。
- ルールキット配布: 「①いらない物を1つ選ぶ ②家族・親戚・友だちの家族と3回まで交換 ③何になったか `#わらしべチャレンジ` で報告」
- 北米で伝説になった「赤いクリップ1つを家1軒まで交換した」実話の家族版。**ゲームの外で遊べるブランド**を作る
- 安全ガイドライン同梱: 交換相手は保護者の知る範囲のみ／金銭・高額品は禁止／学校への持込は先生に確認

## 3.2 展開
1. ファミリー系YouTuber/インフルエンサー5組に先行キット提供（動画映えが構造的に保証されている企画: 過程がドラマになる）
2. アプリ内「うちのわらしべ」ページ(08 D-2)が受け皿
3. 優秀チャレンジは公式が絵巻風イラスト化して表彰（賞金なし・名誉のみ＝健全性維持）
- KPI: 初回3ヶ月でハッシュタグ投稿500件・関連動画総再生100万

## 3.3 なぜこれが効くか
広告は「アプリを知らせる」だけだが、チャレンジは**「交換って面白い」という体験そのものを世の中に配る**。体験した家族にとってアプリは「あの楽しかった遊びの完全版」になる。CACゼロで、模倣困難（後発が同じ企画をやれば本作の宣伝になる)。

# 5. チャネル別プレイブック# 5. チャネル別プレイブック

## 5.1 App Storeフィーチャー（費用ゼロ・効果最大級の穴場）
- 子ども向けカテゴリは高品質アプリが希少でエディトリアルに拾われやすい
- 提出物: フィーチャー申請フォーム＋プレスキット（世界観アート・安全設計一覧・アクセシビリティ対応表・受賞歴）
- 狙う棚: 「今日のApp」より現実的な「子どものためのApp」「新学期特集」「夏休み特集」
- Google Playは「Teacher Approved」プログラム申請を必須タスク化

## 5.2 ショート動画（TikTok/YouTube Shorts/Reels）
- 公式は「ぶんの30秒劇場」: 特産品トリビア＋交換オチの定型フォーマットを週2本（AI動画生成＋DaVinci Resolveの既存スキルで内製）
- 勝ち筋は公式バズではなく**フォーマットの配布**: 「わらしべチャレンジ」「うちの子の初交換リアクション」等、親が撮りたくなる型を提示する側に回る

## 5.3 保護者インフルエンサー
- ティア設計: S=教育系書籍著者・FP系発信者（信頼借用・タイアップ記事）／A=ママ系Instagram 1-10万（ギフトコード提供・PR表記遵守)／B=マイクロ（体験会招待）
- 全提供に共通条件: **正直に書いてもらう**（提灯レビュー禁止を明文化。長期の信頼>短期の絶賛）

## 5.4 受賞・権威戦略
- 応募カレンダー: キッズデザイン賞（5月応募/8月発表）→グッドデザイン賞（翌年4-6月）→日本e-Learning大賞（8-9月）
- 受賞は親・学校・メディア・ストア審査すべてに効く「一石四鳥の資産」。応募書類は本ドキュメントスイートから流用可能（すでに書いてある）

## 5.5 岡山・地域戦略
- 山陽新聞・地元TV・岡山県観光連盟・倉敷デニム/備前焼の組合へ「地元が舞台のアプリ」として持込
- 将来の「ほんものバッジ」（v2.0 §8.4）の交渉先リスト化を今から。地域から全国へ、は教育系の王道の攻め順

## 5.6 やらないことリスト
- 運用型広告への依存（LTV/CAC>4の原則。使うのは検索広告の指名・カテゴリワード少額のみ）
- 子ども向けコミュニティの自社運営（安全コスト過大。コミュニティは親向けのみ）
- 比較広告・競合言及／値引き常態化（ブランドの品位＝親の信頼）

# 7. 失敗パターンの先回り（認知編）

| 失敗パターン | 先回り策 |
|---|---|
| ローンチ日に話題が集中し翌週に消える | 4月・7月・12月の三山構造＋常時UGC（チャレンジ）で「線」の認知にする |
| 教育文脈で紹介され「勉強アプリ」の顔が先に立つ | 子ども向け素材は徹底して冒険の顔。教育の顔は親・学校向け資料に分離（v2.0 §1.3の三つの顔を媒体でも守る） |
| インフルエンサー投稿が広告臭くなる | 現物は「体験会招待＋子どもの実プレイ」中心。台本を渡さない |
| 地方発が「ローカルな話題」で止まる | 岡山はストーリーの起点として使い、訴求の主語は常に「日本の子どもの金融教育」 |
