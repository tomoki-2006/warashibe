# 第0部 エグゼクティブサマリー

**一言で**: 藁一本から交換だけで成り上がる交易アドベンチャー。子どもは「面白いから」遊び、親は「金融教育になるから」払い、学校は「教材になるから」導入する。三者それぞれに別の顔で売る、一つのプロダクト。

**勝利条件**: 「子どもの金融教育といえばわらしべワールド」という第一想起の獲得。英語のDuolingo、算数のトドさんすうに対する、お金・経済カテゴリの空位の玉座に座る。

**戦略の核心（3行）**:
1. 面白さで子どもを掴む（Phase 0-1で証明）
2. 安全性と学習可視化で親を掴む（広告ゼロ・ガチャゼロ・親ダッシュボード）
3. 実績を担いで学校を掴む（桃鉄教育版の後追いではなく、金融教育必修化の本丸を取る）

**数字の骨子（3年後の目標・仮説）**: 累計40万DL / 有料転換4% / 年間経常収益6,000万円 + 教育機関ライセンス。ソロ＋AIエージェント体制のため損益分岐が極端に低く、1万有料会員で十分に事業として自立する。

# 第1部 市場と戦略

## 1.1 追い風（なぜ「今」なのか）

| 追い風 | 内容 | 本作への意味 |
|---|---|---|
| 金融教育の制度化 | 2022年高校家庭科で金融教育必修化。小中への拡大議論が継続 | 「小中学生向けの受け皿」が制度側から求められている |
| 親世代の不安 | NISA拡充・インフレで「子どもにお金の教育をしたいが方法がない」という親の課題が顕在化 | 課金動機が既に存在する。作るのは解決策だけ |
| GIGAスクール端末 | 小中学生ほぼ全員が学習用端末を保有 | 学校チャネルの配布インフラが国費で敷設済み |
| 桃鉄教育版の実績 | 無償提供で小学校約35%（6,800校・2025年3月検証値）に既導入 | 前例で説明コストは下がったが、**学校の価格アンカーは¥0に固定された**。無償で入り、一括でのみ売る（§8.3） |
| AI開発コスト革命 | ソロ＋AIエージェントでゲーム開発が成立する時代 | 教育ゲームの採算ラインが根本から変わった |

## 1.2 競合マップと空白地帯

```
        学習色が強い ▲
                    │
     ちびむすドリル │ トドさんすう
     学習ポスター系 │ シンクシンク
                    │ ワンダーボックス
   ──────────────┼────────────── ▶ ゲーム性が強い
     子ども向け     │
     マネー絵本     │ ★わらしべワールド（空白）
     おこづかい帳   │ 桃鉄（教育は副産物）
     アプリ         │ あつ森（経済要素はあるが教育設計なし）
        娯楽色が強い▼
```

- **【v3.2改訂】「有料×ゲーム性×体系カリキュラム」の3条件同時充足は不在。ただし周辺は無料勢で飽和**（2026年時点・検証済み）: ハロまね（三井住友カード）・まねぶー（スポンサー型）・かぶポン！・うんこお金ドリル（金融庁コラボ）等、金融機関CSR・スポンサーモデルの無料アプリが棚を占有。学校は桃鉄教育版が無償で小学校約35%（6,800校）に既導入、マップ編集「マイ桃鉄」や収益率を扱う授業例まで持つ
- **本作の敵は競合の品質ではなく、価格アンカー¥0である。** よって有料の芯は「ゲームの続き」ではなく「家庭に持ち出せる成果物（行商手帳・自由研究・級）」に置く（08 C群を課金価値の第一列とする）
- 品質側の差別化は不変: おこづかい帳は記録ツールでゲームではない。桃鉄は経済「気分」であって「構造」（価値の主観性・交換の連鎖）を教えない
- 間接競合はシンクシンク/ワンダーボックス（知育サブスクの財布の競合）。差別化は「お金・経済という親の最重要関心テーマ」への特化
- 最大の敵は競合アプリではなく**YouTube/ゲーム一般との可処分時間競争**。だからこそ「1旅15分で完結」「親が時間制限を設定できる」設計が営業上の武器になる

## 1.3 ポジショニング宣言

**「日本一安心して渡せる、日本一夢中になるお金のゲーム」**

- 子ども向けの顔: 「藁から屋敷まで成り上がれるか？」（成り上がり・収集・冒険）
- 親向けの顔: 「遊んだ分だけ、金融リテラシーが貯まる」（安全・可視化・非中毒設計)
- 学校向けの顔: 「社会科×家庭科×道徳をつなぐ横断教材」（地理・経済・共感）

## 1.4 堀（Moat）の設計 — 時間が経つほど強くなる資産

1. **信頼ブランド**: 広告ゼロ・ガチャゼロ・チャットゼロは後発F2P企業が構造的に真似できない（彼らの収益モデルと矛盾する）
2. **コンテンツ資産**: ルートはデータ駆動で量産可能。47都道府県×時代×季節イベントで数百本のルートが積み上がる。後発は同じ物量を追うのに年単位かかる
3. **図鑑・学習履歴のスイッチングコスト**: 子どもの成長記録が溜まるほど移行障壁が上がる
4. **学校導入実績**: 教育チャネルは一度入ると剥がれにくい。先行導入校リストそのものが営業資産
5. **原作がパブリックドメイン**: わらしべ長者はIP使用料ゼロ。かつ日本人全員が知っている＝説明コストゼロの最強IP

## 2.1 ペルソナ（4人）

**① ゆいちゃん（9歳・小4・コアユーザー）**
親のスマホとGIGA端末を使う。マイクラとすみっコぐらしが好き。「お勉強アプリ」は3日で飽きた前歴あり。刺さるポイント: 交換が成立した瞬間の気持ちよさ、図鑑が埋まること、「もう一回」できる短さ。

**② ゆいちゃんの母（38歳・課金決裁者）**
NISAは始めたが自分の金融知識に自信がない。子どものYouTube時間に罪悪感。ママ友のLINEグループが情報源。刺さるポイント: 「今週の学び」レポート、時間制限機能、広告ゼロ。**彼女がApp Storeレビューを書き、ママ友に共有する人。マーケティングの主役。**

**③ 田中先生（45歳・小学校教諭・学校チャネルの鍵）**
桃鉄教育版をクラブ活動で使った経験あり。金融教育を「やれ」と言われているが教材がない。刺さるポイント: 指導案テンプレ付き、1コマ45分で収まる授業モード、成績と紐づかない安心感。

**④ おじいちゃん（68歳・ギフト購入者）**
孫に何か買ってあげたいが、ゲームは母親に嫌がられる。刺さるポイント: 「教育にいいゲーム」という大義名分。**年間プランのギフトコード**が孫へのプレゼントになる。

## 2.2 世界観と物語

- **舞台**: 「どこか懐かしい、江戸と昔話のあいだの日本」。特定の年号を持たない架空の時代（歴史考証の呪縛を回避しつつ、実在の地名・特産品は使う）
- **プレイヤー**: 名もなき旅人。名前はニックネームのみ（個人情報を取らない設計と一体）
- **導きの声**: 各地の「市の神様」。ルート開始と終了に一言だけ語る控えめな存在。説教はしない

## 2.3 相棒キャラクター「ぶん」— 本作のマスコット（新決定）

ストップ1で捕まえたアブが、旅の相棒として肩に留まり続ける。名前は「ぶん」。

- **なぜアブか**: 原作の必然性（わらしべ長者の物語はアブから始まる）。誰も使っていないモチーフで、権利的に完全クリーン。「嫌われ者の虫が最初の価値を生んだ」という物語がゲームのテーマ（価値は見方次第）そのもの
- **機能**: ヒントの発話者（「ぶんぶん…あの子、何を見てるんだろ？」）、ロード画面の遊び相手、結果カードの隅に必ずいる
- **デザイン要件**: 丸く、羽を大きく、リアルな虫感を完全排除。キービジュアル発注時の最重要項目
- **リスクヘッジ**: 子どもテストで「こわい/きもちわるい」反応が2割を超えたら、デザイン再調整 → それでもダメなら雀に交代（判断はPhase 0テストで）

## 2.4 ライティングガイド（全テキスト共通ルール）

1. 小3が音読できる語彙。難語には必ずふりがな＋タップで意味ポップアップ
2. 一文は40字以内。セリフは2文まで
3. 説教・解説を書かない。学びはすべて「体験と星の差」で伝える（価値メーターが先生）
4. NPCは断るときも必ず相手を尊重する（「いらない」ではなく「ありがたいけど、いま欲しいのは…」）— 交渉の礼儀もカリキュラムの一部
5. カタカナ語は使わない（世界観維持。「ヒント」→「ささやき」等)
6. フォントはUDデジタル教科書体系（ディスレクシア配慮・学校導入時の評価点）

# 第3部 ゲームデザイン詳細

## 3.1 コアループの精密化（v1.0からの深掘り）

### 交換の「気持ちよさ」の分解 — Phase 0で磨き込む対象

| 瞬間 | 演出 | 目的 |
|---|---|---|
| 困りごとに気づく | NPCの視線・仕草アニメ（子どもがアブを目で追う） | 「自分で気づいた」感。ここが観察ゲームの快感の源泉 |
| 提案の瞬間 | 差し出すアイテムがふわっと浮く0.5秒の間 | 賭けのドキドキ |
| 成立の瞬間 | 画面が一瞬明るくなる＋「！」＋固有の成立音 | 報酬のピーク。SEは1旅で7回聞いても飽きない音に |
| 価値メーター | 星の差がカラカラと回って開示 | アハ体験。教育装置 |
| 階段演出 | 画面下部の「価値の階段」が1段光って上がる | 進捗の可視化・あと何段でゴールかの期待 |

### 失敗の設計（詰みゼロの先へ）

- 誤提案はペナルティなし、ただし**ぶんが小声でつぶやく**（「ん〜、ちがったみたい」）
- 同じNPCに同じアイテムを2度出せない（総当たり防止 → 観察への誘導）
- ヒント梯子3段目で実質答え。**「わからなくて泣く」子をゼロにするのが絶対条件**（親の解約理由の筆頭は「子どもが詰まって癇癪」）

## 3.3 時代進化システム — 具体メカニクス（本計画書の最重要拡張）

### 時代2: 通貨時代 —「相場」の発見

- 解放条件: ルート8クリア（物々交換の不便を約8旅ぶん体験済み）
- 新メカニクス:
  - **相場ボード**: 各町の市場に「この町での売値・買値」が掲示される。同じ昆布が産地で安く、都で高い
  - **行商モード**: 元手のお金で仕入れ→別の町で売る。差額が利益（安く買って高く売る、を自分の手で）
  - **重さの制約**: 小銭は重い。大金を運ぶと移動が遅くなる＋（時代3への伏線）
- 学び: 価格は場所で違う／お金は「何とでも交換できる万能アイテム」という発明

### 時代3: 銀行・手形時代 —「運ばないお金」

- 解放の物語: 山道で追いはぎイベント（所持金の一部を失う※演出はコミカルに）→ 両替商が「手形」を教えてくれる
- 新メカニクス:
  - **為替手形**: 京都で預けて江戸で引き出す。手数料はかかるが安全
  - **預け入れと利息**: 使わないお金を預けると、旅から帰るたびに少し増えている（時間がお金を生む最初の体験）
  - **借入（上級）**: 大口仕入れのための借金。利息より儲けが大きいときだけ得をする——返済シミュレーション付きでしか借りられないUI
- 学び: 信用が距離を超える／利息は時間の値段／借金は道具（怖さと使い方をセットで）

### 時代4: 投資・株時代 —「助けた店が、あなたの資産になる」（キラー機能）

- 新メカニクス:
  - **のれん分け投資**: 過去のルートで助けたNPCたち（布問屋の娘、備前焼の行商人…）が店を構えている。その店に出資できる
  - 出資した店は日次で配当を生む。店の成長イベント（新商品・支店）で配当が増える
  - **リスクイベント**: 火事・不作が特定の店を襲う。1店集中のプレイヤーは大打撃、分散していれば軽傷 → **分散投資を体で学ぶ**
- 感情設計: 投資先が「知らない会社」ではなく「自分が旅で助けた、顔の見える相手」であること。投資=応援という原体験を刷り込む。これは大人の金融教育アプリにも存在しない、本作だけの発明
- 学び: 資産は寝ている間も働く／分散／キヨサキのキャッシュフロー概念のゲーム化（v0.1で決めたキャップストーンの実装形）

### 時代5: 信用時代 —「見えない資産」

- 新メカニクス: 取引履歴から「信用度」が育つ。高信用者は掛け取引（後払い）や大口取引が解放。約束を破る選択肢も提示され、破ると一瞬得するが信用度が下がり長期で大損する構造
- 学び: 信用は最も高価な資産／目先の得と長期の得

### 時代と地理の重ね合わせ（v0.1決定の実装方針）

共有ワールドでは地図上に全時代が同居する。田舎=物々交換、城下町=通貨、湊町=手形、都=株式。プレイヤーは自分の理解度に合った土地で暮らせばよく、**「ついていけない子」が構造的に発生しない**。

## 3.4 図鑑システム詳細（Phase 1）

- 三部構成: **特産品図鑑**（産地・豆知識・実物写真への切替ボタン付き※現実接続）／**人物図鑑**（職業解説: 行商人とは、問屋とは、両替商とは）／**土地図鑑**（行った町の実在情報）
- 完成度は都道府県単位で%表示（「岡山 80%」）→ コンプ欲と地理学習が直結
- レアリティは3段階のみ（ふつう/めずらしい/おたから）。**入手条件は全開示（確率要素なし・完全決定的）。入手手段はプレイのみ**
- 「実物写真への切替」が図鑑の隠し玉: ゲーム内アイコン⇄本物の備前焼の写真をタップで切替。ゲームと現実が接続する瞬間を作る（親が最も感動する機能と予測。家族旅行の行き先候補にもなる → 口コミの種）

## 3.5 デイリー行商・イベント（Phase 1-2）

- **デイリー行商**: 1日1回・5分・1〜2交換の小クエスト。報酬は図鑑の欠けピース
- **のれん（streak）**: 連続日数で店の暖簾が育つ。切れても翌日から続きが育つ「破れない暖簾」設計（Duolingoのstreak不安を除去した優しい版。親向けの差別化説明にも使う）
- **季節イベント**: 正月の初売りの旅、夏祭りの旅、秋の収穫の旅。年中行事×経済（実売期のASO・復帰導線を兼ねる）

## 3.6 エンドコンテンツ構想（Phase 3-4・共有ワールド）

- **市場掲示板**: 非同期のオファーボード（「昆布1つ ⇄ 塩2つ 求む」）。チャット不要・定型文のみ・承認制
- **世界の時代進行**: 全プレイヤーの経済活動の総量で世界の時代メーターが進む。「みんなで文明を進めている」感
- **長者番付**: 資産額ではなく**「助けた人の数」「交換の巧さ」の番付**を筆頭に置く（拝金ゲーム化の防止・親向けの倫理的差別化)

## 6.4 事業主体（v3.2新設・Phase 0中に確定）
