# 3. セリフ執筆規程（ライティング6原則の運用版）

**NPC共通**: 一文40字以内・セリフ2文まで／困りごとの核心は intro で「音読しない」（様子・状況・observable に散らす）／断り文は必ず相手を尊重（L1に方向ヒントを内包）／カタカナ語禁止（許可リスト: なし）。

**ぶんの口調規程（決定42・これが正）**:
- 一人称は「ぼく」。敬語なし・語尾は伸ばさない
- 口癖「ぶんぶん」は**1発話に最大1回**（文頭か文末のどちらか）。多用すると鬱陶しさが先に立つ
- 感嘆符は1発話に「！」1つまで
- **答えを言っていいのはL3だけ。** L2までは視点の誘導のみ（「〜を見てみよ？」「〜って言ってたね」）
- 悲しい場面・真剣な場面では口癖と「！」を使わない（羽音は静かになる）
- ぶんは賢いのではなく「よく見ている」。知識を披露するセリフは書かない（それは図鑑の仕事）

# 4. 記入例（確定企画）: ルート2「せとうちの旅」

```
■ ルート名: せとうちの旅
■ 地方・実在地名: 瀬戸内海（塩田・小豆島・島々の湊）
■ 学習テーマ: 海運と物流 —「はこぶこと」そのものが価値になる
■ 交換チェーン:
  ①わら → ②わらぞうり（合成） → ③しお → ④ひもの → ⑤そうめん →
  ⑥ふねのつな → ⑦のりふだ（乗船札） → ⑧くらのかぎ（蔵の鍵）
■ 合成レシピ: わら × わら ＝ わらぞうり（S0で2本目のわらを拾う）
■ 複数解ストップ: S2, S5
■ 感情ピーク: S4 — 嵐のあとの綱問屋。流された店先をいっしょに片付ける（タップ小イベント）
■ ストップ表:
  S0 みなとまち     | 老船頭   | 開始。「海の旅は、陸とは荷の運び方がちがうぞ」＋わら2本目
  S1 えんでん(塩田) | 塩職人   | 困り: 素足が焼けて作業が進まない（observable: 赤い足あと）
                    | 正解: わらぞうり → しお | 豆知識: 瀬戸内は日本の塩どころ
  S2 しまのひものごや【複数解】| 干物職人 | 困り: 獲れた魚を長持ちさせたい
                    | 正解A: しお → りっぱな ひもの ／ 正解B: わらぞうり → こぶりの ひもの
                    | 学び: 相手の一番の困りごとを解くほど、見返りは大きい
  S3 しょうどしま   | そうめん屋 | 困り: 昼を食べそこねて延ばす手が止まる（observable: からのどんぶり）
                    | 正解: ひもの → そうめんのたば | 豆知識: 小豆島は400年のそうめんどころ

                    | 片付けイベント → そうめん → ふねのつな | 学び: 助け合いも交換のうち
  S5 おおきな帆船【複数解】| 船長 | 困り: 出航前、綱が古い。航海の食料も心もとない

                    | 手伝いイベント（縄の結びの腕を見せる）→ のりふだ半券＋働きぶり → くらのかぎ
                    | 締め: 最後の交換は「物」ではなく「腕と信用」——10章 §1.3の先取り
■ 図鑑新規: アイテム6点 / 人物7人 / 土地5箇所
```
