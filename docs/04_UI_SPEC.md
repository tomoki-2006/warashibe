  --fs-body: 18px; --fs-serif-name: 20px; --fs-btn: 20px;
  --fs-title: 32px; --fs-small: 14px;

# 2. 画面レイアウト（縦持ちスマホ390×844基準・タブレットは中央840px幅に収めて余白）

## S2 マップ（絵巻）
```
┌─────────────────────────────┐
│ [ちず全体]              [にもつ]│ ← 上部バー(高さ64)
│                               │
│   ～～絵巻横スクロール～～      │
│  🏔️  ⛩️   🍵  🏘️  ⚓  🏮  🏯 │ ← ストップアイコン(72px)
│      ▲現在地(旅人+ぶん)        │
│                               │
│ ┌───────────────────────┐ │
│ │ 🌾→🧸→🍡→＿→＿→＿→＿→🏠 │ │ ← 価値の階段(下部固定・高さ88)
│ └───────────────────────┘ │
└─────────────────────────────┘
```
- スクロール: 横スワイプ＋解放ストップのタップで自動歩行。未解放は靄＋鍵なし（存在だけ見せ期待を作る）
- 価値の階段: 現在アイテムまで実絵文字、先は「＿」。ゴール🏠は常時表示。段はgives連鎖の序数（弱った馬は半段点滅・v3.2）

## S3 会話
```
┌─────────────────────────────┐
│            [背景イラスト]      │
│        [NPC立ち絵 320px]      │
│  ┌─────────────────────┐  │
│  │ 泣いている子             │  │ ← 名前札
│  │ 「うえーん！ うえーん！」  │  │ ← セリフ枠(最大2行・タップ送り)
│  └─────────────────────┘  │
│ [はなす] [しつもん] [こうかん]  │ ← 行動バー(各ボタン--tap-min)
│                     [にもつ]  │
└─────────────────────────────┘
```
- セリフ送り: 枠タップ。文字送りアニメ25ms/字（タップで即全表示）
- TTS: 設定ON時、セリフ表示と同時に読み上げ（Web Speech API・rate 0.9）
- しつもん: 押すと質問リスト（最大2件・使用済みは消える）

## S4 にもつ（モーダル）
- 3列グリッド・枠96px・絵文字64px＋名前ルビ
- 選択で詳細カード（産地・豆知識）。2つ選択で[くみあわせる]活性化
- こうかんフローから開いた場合はタイトルが「なにを さしだす？」に変化＋提示済みアイテムはグレーアウト

## S6 価値メーター（成立演出内）
```
   ┌───────────────────┐
   │      きびだんご 🍡      │
   │ あなたにとって ★★☆☆☆ │
   │ あいてにとって ★★★★★ │
   │ 「はらぺこには なにより  │
   │   の ごちそう！」        │
   └───────────────────┘
```
- 星は左から順に150ms間隔で回転出現。相手側の5つ目で朱色フラッシュ＋SE

## S7 リザルト
- 成長チェーンアニメ→暖簾（ランク色）が下りる→結果カードプレビュー→[もういちど][みんなにみせる]
- 「みんなにみせる」は**親ゲート内設定で有効化した端末のみ表示（既定OFF・v3.2）**。有効時もOS共有シート起動のみ（審査解釈のブレを排除）

# 4. アニメーション数値表（Framer Motion実装値）

| 演出 | 時間 | イージング | 備考 |
|---|---|---|---|
| 提案アイテム浮遊 | 500ms | --ease-out | y:-24px・影拡大 |
| 成立フラッシュ | 300ms | linear | 白→透明オーバーレイopacity .6→0 |
| 「！」ポップ | 400ms | --ease-spring | scale 0→1.2→1 |
| 星出現(1個) | 220ms | --ease-spring | 回転180°＋scale |
| 階段1段上昇 | 600ms | --ease-spring | 新絵文字がバウンス着地 |
| 拒否の首ふり | 350ms | ease-in-out | NPC立ち絵 x:±6px×2 |
| ぶん吹き出し | 250ms | --ease-out | 下から16px |
| 画面遷移 | 320ms | --ease-out | クロスフェード＋4pxスライド |
| 合成演出 | 1200ms | 専用 | 2アイテム衝突→光→新アイテム |
| チェーン最終アニメ | 260ms/項目 | --ease-spring | 左から順に点灯・計約2.5s |

- 全演出は`prefers-reduced-motion`で時間半減＋バウンス無効（アクセシビリティ）

# 5. 音仕様（アセットリスト）

| ID | 用途 | 長さ | トーン指示 |
|---|---|---|---|
| se_accept | 交換成立 | 0.8s | **最重要。** 鈴＋琴のアルペジオ上昇。3案発注し子どもテストでA/B |
| se_decline | 丁寧な拒否 | 0.4s | 低い木魚系・ネガティブすぎない |
| se_stair | 階段上昇 | 0.5s | 音階が旅の進行で1音ずつ上がる（6段＋クリア和音で1オクターブ完成・v3.2） |
| se_get | アイテム入手 | 0.3s | ポン(鼓) |
| se_bun | ぶん登場 | 0.3s | 羽音を可愛く加工 |
| bgm_michi | 道中 | 90sループ | 篠笛×ローファイ・音量-14LUFS |
| bgm_ichi | 出会い/市 | 60sループ | 三味線ピチカート |
- 実装: 全SEはスプライト1ファイル(howler.js)。BGMはユーザー初タップ後に再生開始（自動再生制限対応）

# 6. アクセシビリティ要件（受入基準に含む）

- 全タップ対象 ≥56px／フォーカス可視化（GIGA端末のキーボード操作対応）
- コントラスト比: 本文4.5:1以上（生成り背景×墨でクリア済）
- 全画像に代替テキスト／TTS全文対応／時間制限なし／点滅3回/秒以下
